# Blog Phase 13: Detail Body Softening

## 目标
- 完成 Blog 详情页视觉迁移的 Phase 2。
- 让正文区、文中图片和页尾从旧的“文章卡片感”过渡到更像首页延伸出来的轻量阅读布局。

## 改动
- 去掉正文各 section 之间原有的硬分隔线，改为更依赖留白组织层级。
- 微调正文段落、列表和小标题节奏，让阅读流更连续。
- 去掉正文图片外层的卡片边框、背景和内边距，只保留更直接的圆角图片展示。
- 将详情页 footer 从独立卡片改为更轻的尾注样式，只保留顶部细线。

## 验证结果
- `pnpm typecheck` 通过
- `pnpm lint` 通过
- 用 headless Chrome 检查了：
- `/blog/communication-at-work` 的桌面端和移动端
- `/blog/smartx-design-workflow` 的桌面端
- 结果确认：正文区已经不再像一段段卡片拼接，图片和页尾也与新的头部语言更一致
