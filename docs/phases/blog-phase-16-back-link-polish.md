# Blog Phase 16: Back Link Polish

## 目标
- 只优化 Blog 详情页左上角 `Back` 的视觉样式与位置。
- 让它更接近参考页面的轻量胶囊按钮语气。

## 改动
- 去掉 `Back` 外层多余的左侧内边距，让按钮更贴近详情页外框起点。
- 将按钮背景、描边和文字颜色收回到更轻的中性色层级。
- 补齐更克制的 `hover` 与 `active` 状态，避免只改字色导致交互反馈太弱。

## 验证结果
- `pnpm typecheck` 通过。
- `pnpm lint` 通过。
- 使用 Playwright CLI 抓取了详情页默认态截图，确认返回按钮的位置和默认样式已经更接近参考图。
- Playwright MCP 传输仍然断开；hover 自动截图脚本因为当前临时 CLI 运行时没有暴露 `playwright/test` 模块而未完成，但默认态视觉和 hover CSS 已核对。
