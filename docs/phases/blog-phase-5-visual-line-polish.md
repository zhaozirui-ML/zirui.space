# Blog Phase 5

## 目标

对 `/blog` 列表页做一轮视觉精修，重点统一页面中的线条语言，让不同层级的结构分别使用更合适的实线、虚线和线强度，而不改变当前页面布局和内容结构。

## 改动

- 更新 `src/site/styles/blog-page.module.css`，把页面边框拆成 4 个层级变量：基础实线、强调实线、虚线分隔线、弱化辅助线。
- 保留区块外框与主骨架为实线，让整体结构仍然稳定。
- 将标题区和区块内容之间的分隔调整为虚线，并给 `Featured` / `Browse all` 标题增加两侧短虚线，强化标题的居中感和编辑排版气质。
- 为 `Featured` 区内部卡片、`Browse all` 网格内部卡片增加虚线分隔，减少原本“所有边框都一样”的单调感。
- 为卡片图片补充轻量内描边，让图片和暖白背景之间的边界更清晰。

## 验证结果

- `pnpm typecheck` 通过。
- `pnpm lint` 通过。
- 使用 Playwright CLI 检查了 `/blog` 的桌面端与移动端页面，确认：
  - 实线主要保留在页面外框和主结构边界。
  - 虚线主要用于区块标题、内容分隔与网格内部分界。
  - 移动端没有出现列分隔线残留或错位。

## 下一步

- 如果你希望继续往更“杂志化”的方向推进，可以下一步再精修标题字重、日期颜色和卡片裁切比例。
- 如果你觉得现在虚线数量偏多，也可以再做一轮减法，只保留标题区和卡片区两类虚线。
