## Users

这个作品集的主要受众包括：

- 招聘经理
- 设计经理
- 同行设计师

他们在浏览时，需要尽快判断下面三件事：

- 是否兼具产品设计与前端实现能力
- 是否适合高级产品设计岗位
- 是否能够处理复杂系统，并把复杂问题讲清楚

因此，这个网站不应该只展示“做过什么”，还应该帮助受众快速感知：

- 判断力
- 系统思维
- 交付能力
- 设计与实现之间的一致性

## Brand Personality

这个作品集的核心气质是：

- 冷静
- 可信
- 细腻

补充人格特征：

- 聪明
- 有判断力
- 温和

文案气质也应遵守同一套判断：

- 中英文文案不追求字面形式完全一致，而要追求气质一致
- 文案调整时，优先选择各自语言里最自然、最得体、最符合页面气质的表达
- 如果英文使用省略号、短句停顿或更轻的语气，不代表中文必须机械复刻相同标点
- 如果中文用句号更稳、更克制、更像作品集表达，应优先保留中文自己的自然节奏
- 判断标准不是“看起来是否完全对称”，而是两种语言是否都传达出同样的冷静、可信、细腻与克制
- 后续所有文案微调，都应按“各自语言最自然的方式统一气质”这个原则处理

整体表达应避免：

- 过度自我宣传
- 过度表演化
- 追逐短期视觉趋势
- 用情绪化视觉掩盖信息结构

## Aesthetic Direction

当前项目已经建立的方向应继续保留：

- 精致
- 克制
- 内容优先
- 叙事感
- 温暖留白
- 轻 editorial 气质

参考方向：

- [instruct.ai](https://instruct.ai/)
  主要参考其设计系统气质：小清新、精致、优雅
- [jaffe.page](https://www.jaffe.page/)
  主要参考其人文感配色、叙事感和细节处理
- [nelson.co](https://nelson.co/)
  主要参考其克制感与微动效的分寸

明确避免的方向：

- 过度科技感
- 过度后台感
- 过度插画化
- 过度依赖强品牌色
- 为了“显得高级”而堆砌重动效或重装饰

## Theme Strategy

当前实现仍然是 light-first，但 dark mode 已经进入近期接入范围。

- 现阶段浅色主题是主要基线
- 已准备接入 dark mode
- dark mode 不应该走霓虹、赛博、重科技路线
- 暗色主题仍应保留当前项目的冷静、可信、细腻、克制与叙事感

接入 dark mode 时，优先从 token 和主题层扩展，不要直接在页面组件里零散补色值。

## Design Principles

1. 先展示高级判断力，再展示视觉表现力。
   页面首先要帮助受众理解你的思考质量、系统能力与成熟度，而不是先看到炫技。

2. 用设计解释复杂性，而不是装饰复杂性。
   版式、层级、留白、交互都应帮助读者更快理解复杂系统。

3. 保持温和、克制、可信的人文感。
   网站应显得有分寸、有修养，而不是吵闹、强推销或过度品牌化。

4. 主要靠排版、节奏和细节建立高级感。
   优先使用字体、间距、信息层级和图文关系建立品质感，而不是依赖强装饰。

5. 动效必须服务阅读与理解。
   动效应轻、准、克制，只帮助建立节奏、焦点和过渡，不抢内容。

6. 设计与代码要共同传达同一个结论。
   视觉质量和前端实现质量应一起证明：你不仅能提出方案，也能把体验真正做出来。
